## ✨ لماذا "صلاتي"؟
ركزنا في هذا المشروع على **السهولة المطلقة**؛ لا إعدادات معقدة، لا استهلاك للبطارية، فقط ما تحتاجه في لحظات.

*   **⚡ تثبيت فوري (PWA):** لا حاجة لمتجر التطبيقات. ثبته مباشرة من المتصفح ليعمل كتطبيق منفصل بلمسة واحدة.
*   **🎨 واجهة مريحة للنظر:** ألوان مستوحاة من **Material Design 3** توفر راحة بصرية في الليل والنهار.
*   **🌍 دقة عالمية:** أينما كنت، يحصل التطبيق على مواقيت الصلاة الدقيقة لمنطقتك فوراً.
*   **📅 تقويم متكامل:** عرض الوقت بنظام 12 ساعة مع التاريخ الهجري والميلادي محدثاً بدقة.
*   **🚀 أداء فائق الخفة:** مبني بـ **Vanilla JS**، مما يعني سرعة استجابة مذهلة واستهلاك بيانات شبه منعدم.

## 🛠 التقنيات المستخدمة
المشروع يعتمد على تقنيات الويب الحديثة لضمان أقصى توافقية:
*   **الأساس:** HTML5 & CSS3 (Modern Flexbox/Grid).
*   **المنطق البرمجي:** Pure JavaScript.
*   **البيانات:** Aladhan API لمواقيت دقيقة عالمياً.
*   **الخصائص:** Manifest API لتمكين خاصية التثبيت.
